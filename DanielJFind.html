Método Find

Este método en JavaScript se utiliza para buscar el primer elemento en un array que cumpla con una condición especificada en una función de prueba. Este método es útil cuando necesitas encontrar un elemento específico dentro de un array basado en ciertos criterios de búsqueda.

Por ejemplo:

  1. Juan
  2. Pedro
  3. Frank
  4. Pim
  5. Kanye
