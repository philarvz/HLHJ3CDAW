¿Qué es sort()?

Explicación para niños: Imagina que tienes una caja llena de juguetes y quieres ordenarlos del más pequeño al más grande. sort() es como un ayudante mágico que te ayuda a hacer eso, pero con números o palabras en una lista.

Explicación técnico: El método sort() en JavaScript se utiliza para ordenar los elementos de un array. Por defecto, los elementos se ordenan como strings en orden lexicográfico. Para ordenar números correctamente, se debe proporcionar una función de comparación.

El término "lexicográfico" se refiere a un método de ordenación que sigue el mismo principio que el orden alfabético en un diccionario. En el contexto de la programación y, más específicamente, en JavaScript, el orden lexicográfico se utiliza para describir cómo se comparan y ordenan las cadenas de texto (strings).
